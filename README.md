🏨 Sanjay Motel PVT LTD  – Landing Page
A premium, high-conversion landing page for a luxury hotel and resort. This project features a modern UI, booking call-to-actions, and a fully responsive layout.

✨ Key Features
Booking Integration: Clear, accessible call-to-action buttons for reservations.

Hero Section: High-impact visual display with smooth scrolling animations.

Room Gallery: Organized display of luxury suites and amenities.

Testimonials: Built-in section for guest reviews and social proof.

Contact & Location: Integrated map view and contact form for inquiries.

🛠️ Tech Stack
Frontend: React.js / HTML5 / CSS3

Styling: CSS Modules / Tailwind CSS (Optional)

Animations: Framer Motion / AOS Library

Deployment: Vercel
